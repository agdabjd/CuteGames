# Cute Games

Projeto desenvolvido em java utilizando a API Swing para criar as interfaces, com o objetivo de estudar e por em prática os conhecimentos adquiridos sobre a linguagem.

## Jogos que serão desenvolvidos:
- Jogo da advinhação
- Forca
- Jogo da Velha
- Sudoku
- Wordle
- Perguntas e respostas de matemática básica
